##  This program takes a user loaded image and applies 5 randomly placed alterations to it. The user then has to spot the differences.
  3 wrong clicks and the game is over. If the user finds all 5 differences they win.  
  To run the program, the following will have to be installed on the users computer:
- opencv-python
- Pillow
- numpy
###  To install these, run:
  
```bash
pip install opencv-python pillow numpy
```
 or:
 ```bash
 pip3 install opencv-python pillow numpy
 ```

## To run the program:

```bash
python Question_1.py
```
or:
```bash
pyhton3 Question_1.py
```
